### Installation

- Just, Clone this repository -
  ```
  git clone --depth=1 https://github.com/darklusifer/pycript.git
  ```

- Then native the directory -
  ```
  cd PyCript
  ```

- Finally, run the code -
  ```
  python PyCript.py
  ```
